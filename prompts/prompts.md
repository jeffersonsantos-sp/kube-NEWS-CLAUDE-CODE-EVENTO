#Criar Dockerfile

Essa aplicação precisa ser executada utilizando o container Docker.
Proximo passo é fazer a criação do Dockerfile e Docker Compose.
- Analise o projeto para entender as tecnologias e dependencias
- Dockerfile deve seguir as boas praticas
- Docker compose será utilizada para o desenvolvimento

Boas práticas.
- Não utilizar muiti stage build em projeto nodejs
- No Docker compose sempre que utilizar volumes,utilize bind mount para a pasta  .docker_vol na raiz do projeto
